# 聚会游戏合集

线下聚会无纸化游戏房间工具。

## 当前 v0.1 功能

- 首页创建房间 / 输入房间码加入
- 房间页展示房间码、玩家列表、游戏切换
- 狼人杀开局助手静态版
- 其他游戏占位面板

## 技术栈

- React
- Vite
- react-router-dom
- 普通 CSS
- localStorage

## 当前边界

- 暂未接后端
- 暂未接 Supabase
- 暂无真实多人同步
- 暂无登录
- 暂不做线上狼人杀技能、线上投票和自动胜负判断

## 本地运行

```bash
npm install
npm run dev
npm run build
```

## GitHub Pages

推送到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages（仓库需开启 Pages，Source 选择 GitHub Actions）。

## 安全与内容边界

- 不鼓励危险惩罚
- 不强迫饮酒
- 不设计侵犯隐私、羞辱或让人不适的玩法
- 线下玩法应尊重参与者意愿
